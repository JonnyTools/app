# Tech Stack

## Empfehlung

```text
Mobile App:
Flutter

Backend:
Supabase oder eigener Node/Python-Backend

Datenbank:
PostgreSQL

PDF-Verarbeitung:
Python-Service

E-Mail:
Resend, SendGrid, Mailgun oder SMTP

Dateispeicher:
Supabase Storage oder S3-kompatibler Speicher
```

## Warum Flutter

- eine Codebasis für iOS und Android
- gute UI-Möglichkeiten
- stabile mobile Entwicklung

## Warum PostgreSQL

- relationale Daten passen gut zu Aufgaben, Mitgliedern, Ausführungen und Berichten
- gute Abfragen für Analyse
- starke Datenintegrität

## Warum Python für PDF

- gute Bibliotheken für PDF-Text und Tabellen
- später OCR möglich
- Bild-/Farbanalyse möglich
