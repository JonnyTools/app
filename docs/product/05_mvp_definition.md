# MVP-Definition

## Ziel des MVP

Die erste Version soll die Kernlogik der WG App beweisen:

- faire Aufgabenverwaltung
- einzelne Ausführungserfassung
- Preisberechnung
- unterschriebene WG-Beschlüsse
- Monatsbericht

## Enthalten im MVP

```text
Mitgliederverwaltung
Aufgabenverwaltung
Preisverwaltung
Zuständigkeiten
Geteilte Aufgaben
Aufgabe als erledigt markieren
Zusatzaufgabe erfassen
Ersatzaufgabe erfassen
Vorschlags- und Signatursystem
Monatsbericht
Basis-Analyse
Audit-Log
```

## Nicht zwingend im MVP

```text
Automatische PDF-Erkennung mit Farben
Komplexe OCR
Push-Benachrichtigungen
Zahlungsintegration
Foto-Beweise
```

## MVP-Erfolgstest

Der MVP ist erfolgreich, wenn eine WG für einen Monat alle Aufgaben erfassen, prüfen und einen fairen Bericht erstellen kann.
