# Changelog

Alle wichtigen Projektänderungen werden hier dokumentiert.

```text
## Version
Datum

### Hinzugefügt

### Behoben

```

```text

2026-06-06

### Hinzugefügt
- erstes Datenmodell
- Mitgliederverwaltung
- Aufgabenausführung
```
